# Foundational Theoretical Contributions 1875–2025: Full Candidate Log

Filter criteria:
- ✅ Experiment result that forces new axiomatics → in
- ✅ New axiom that establishes or reshapes a field → in
- ❌ Experiment that decides between existing interpretations → out
- ❌ Engineering/technique built on prior theoretical primitive → out

## KEPT (102 entries)

### Physics (33)

| Year | Contribution | Reasoning |
|------|-------------|-----------|
| 1877 | Boltzmann — Statistical Mechanics (S = k ln W) | New axiom: macroscopic thermo emerges from microscopic statistics. Founded statistical mechanics. |
| 1887 | Michelson-Morley — No Aether | Null result that *forced* new axiomatics. Without it no SR. Passes filter: experiment → new axiom required. |
| 1900 | Planck — Quantum Hypothesis (E=hν) | Energy is discrete. New ontological primitive. Founded quantum theory. |
| 1905 | Einstein — Special Relativity | New axiomatics for space and time. |
| 1905 | Einstein — Photoelectric Effect (light quanta) | Light is particulate. Separate axiom from Planck — applies quantization to free radiation. |
| 1905 | Einstein — Brownian Motion (atoms are real) | Statistical prediction of observable fluctuations from atomic hypothesis. New formal framework, not just confirmation. |
| 1913 | Bohr — Quantum Atomic Model | Quantized orbits. First quantum model of matter. Bridge between Planck/Einstein and full QM. |
| 1915 | Einstein — General Relativity | Gravity is spacetime curvature. New axiomatics for gravity, space, time, and matter. |
| 1915 | Noether — Noether's Theorem | Every continuous symmetry ↔ conservation law. New structural axiom connecting symmetry and physics. |
| 1917 | Einstein — Relativistic Cosmology | The universe itself as a dynamical object governed by field equations. New conceptual primitive: cosmology as physics. |
| 1924 | Bose — Bose-Einstein Statistics | Identical particles literally lack identity. New ontological primitive. Not derivable from classical stats. |
| 1924 | de Broglie — Wave-Particle Duality | Matter has wavelength. Extended quantization from radiation to all matter. |
| 1925 | Heisenberg — Matrix Mechanics | First complete formulation of QM. Observables are matrices. |
| 1925 | Pauli — Exclusion Principle | No two fermions in same state. Independent axiom — not derivable from other QM postulates. |
| 1926 | Schrödinger — Wave Equation | Second complete formulation of QM. Dynamics via differential equation. |
| 1926 | Fermi-Dirac Statistics | Half-integer spin particles obey exclusion. Two fundamentally different kinds of particle in nature. |
| 1926 | Born — Probabilistic Interpretation of ψ | |ψ|² = probability. Arguably the most radical single axiom in QM. Without it Schrödinger's equation is just math. |
| 1927 | Heisenberg — Uncertainty Principle | Fundamental limit on simultaneous knowledge. New epistemological axiom. |
| 1927 | Lemaître — Expanding Universe | Theoretical prediction from GR, before Hubble's observation. Cosmology's central axiom. |
| 1928 | Dirac — Relativistic QM / Antimatter predicted | Unified QM + SR. Predicted antimatter from pure theory. |
| 1932 | von Neumann — Mathematical Foundations of QM | Rigorous Hilbert space formulation. Made QM axiomatic. |
| 1934 | Fermi — Beta Decay / Weak Interaction | First theory of a new fundamental force. Not derivable from QED. |
| 1935 | EPR — Entanglement framework | Formalized nonlocal correlations as a theoretical object. |
| 1935 | Yukawa — Meson Theory / Nuclear Force | Predicted force carrier for strong nuclear interaction. New primitive for short-range forces. |
| 1948 | Feynman/Schwinger/Tomonaga — QED | Quantum field theory made consistent. New formal framework: particles as field excitations + renormalization. |
| 1954 | Yang-Mills — Gauge Theory | Non-abelian gauge symmetry. Formal framework underlying all modern particle physics. |
| 1961 | Gell-Mann — Eightfold Way / Quarks | New substructure of matter. Hadrons aren't fundamental. |
| 1964 | Bell — Bell's Theorem | Proved no local hidden variable theory can reproduce QM. New formal constraint on all physical theories. |
| 1964 | Higgs/Englert-Brout — Higgs Mechanism | Spontaneous symmetry breaking gives mass. New mechanism, not derivable from prior gauge theory alone. |
| 1965 | Penrose — Singularity Theorem | Singularities are generic in GR, not artifacts. Changed the meaning of the theory. |
| 1967 | Weinberg-Salam — Electroweak Unification | Unified electromagnetic and weak forces. New structural reduction. |
| 1974 | Bekenstein-Hawking — Black Hole Thermodynamics | Entropy ∝ area, not volume. Connected gravity, QM, thermo, information. New primitive still not fully understood. |
| 1975 | Wilson — Renormalization Group | Universality: different systems behave identically near phase transitions. Explains *why* renormalization works. |
| 1980 | Guth — Inflationary Cosmology | Exponential early expansion solves horizon/flatness problems. New dynamical axiom for cosmology. |
| 1997 | Maldacena — AdS/CFT | Gravity in N dimensions ≡ non-gravity theory in N-1 dimensions. Holographic duality. New axiom-level insight. |

| 1960 | Grothendieck — Schemes | Refounded algebraic geometry on new axiomatics. |

| 1967 | Langlands — Langlands Program | Grand unification conjecture connecting number theory, geometry, and representation theory. |
| 1975 | Mandelbrot — Fractal Geometry | New geometric primitive: fractional dimension. Objects between dimensions. |
| 1982 | Thurston — Geometrization Conjecture | Every 3-manifold decomposes into 8 geometries. New framework for topology. |
| 1993 | Wiles — Fermat's Last Theorem | Proof, but the modularity techniques are foundational. Connected elliptic curves to modular forms. |
| 2003 | Perelman — Poincaré Conjecture proof | Used Ricci flow in foundationally new way. Proved Thurston's geometrization. |
| 2006 | Voevodsky — HoTT / Univalent Foundations | New foundation for mathematics. Types replace sets. Identity has structure. |

### Logic / Computation / Information (19)

| Year | Contribution | Reasoning |
|------|-------------|-----------|
| 1879 | Frege — Predicate Logic (Begriffsschrift) | Invented formal logic. The notation and framework everything else builds on. |
| 1902 | Russell — Russell's Paradox | Broke naive set theory. Forced axiomatization of foundations. |
| 1910 | Russell & Whitehead — Principia Mathematica | Attempted to ground all math in logic. Shaped the foundations program. |
| 1931 | Gödel — Incompleteness Theorems | Any consistent formal system strong enough for arithmetic is incomplete. Foundational limit on formal systems. |
| 1936 | Turing — Turing Machine / Computability | Formalized computation. Defined what is and isn't computable. |
| 1936 | Church — Lambda Calculus / Church-Turing Thesis | Independent formalization of computability. Equivalence with Turing = the thesis. |
| 1939 | Turing — Oracle Machines | Relative computability. New axis: degrees of unsolvability, arithmetic hierarchy. Not an extension of 1936 — a new dimension. |
| 1943 | McCulloch-Pitts — Formal Neuron Model | First formal model of neural computation. New primitive linking logic and biology. |
| 1948 | Shannon — Information Theory | Entropy, channel capacity, coding. Axiomatized information as a measurable quantity. |
| 1949 | von Neumann — Self-Reproducing Automata | First formal theory of self-reproduction. Construction universality. New axis beyond Turing machines. |
| 1950 | Turing — Formal Framing of Machine Intelligence | Not "can we build one" but "what would it mean." New epistemological primitive for an entire field. |
| 1956 | Chomsky — Formal Grammars / Hierarchy | Hierarchy of formal languages. Founded computational linguistics and formal language theory. |
| 1958 | Curry-Howard — Propositions as Types | Proofs = programs. Unified logic and computation at the foundational level. |
| 1965 | Kolmogorov/Chaitin — Algorithmic Information Theory | Complexity of a string = length of shortest program. New formal primitive for randomness and information. |
| 1971 | Cook — NP-Completeness / P vs NP | Formalized computational hardness classes. Central open question of CS. |
| 1971 | Vapnik-Chervonenkis — VC Dimension | Axiomatized what it means for a function class to be "learnable." Formal primitive for learning capacity. |
| 1976 | Diffie-Hellman — One-Way Functions / Public-Key Crypto | New mathematical object class: functions easy to compute, hard to invert. Founded modern cryptographic theory. |
| 1981 | Feynman — Quantum Computing | Proposed that quantum systems require quantum simulation. New computational primitive beyond Turing. |
| 1984 | Valiant — PAC Learning | Formal definition of learnability. "Probably approximately correct." Founded computational learning theory. |
| 1994 | Shor — Shor's Algorithm | Proved quantum computers can factor in polynomial time. Separated BQP from BPP (conditionally). |
